# Analiza Szeregów Czasowych (Time Series Analysis)

Folder zawiera projekty dotyczące analizy danych zależnych od czasu, zrealizowane w języku R.

## Zakres projektów:
1. **Sprawozdanie 1:** - Testowanie białoszumowości (testy graficzne i formalne Ljung-Box).
   - Dekompozycja szeregów czasowych (metody klasyczne, STL, regresja TSLM).
2. **Sprawozdanie 2:**
   - Analiza stacjonarności i różnicowanie szeregów.
   - Identyfikacja modeli AR/MA/ARIMA.
   - Estymacja parametrów (Yule-Walker vs MLE) oraz prognozowanie długoterminowe.

## Wykorzystane pakiety:
- `forecast`, `tseries`, `ggplot2`, `tseries`.
