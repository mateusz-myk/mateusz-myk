# Eksploracja Danych i Uczenie Maszynowe (Machine Learning)

Ten projekt przedstawia porównanie skuteczności różnych algorytmów klasyfikacji w środowisku R.

## Kluczowe zagadnienia:
* **Modele klasyfikacyjne:** Implementacja i tuning parametrów dla k-Nearest Neighbors (k-NN), Drzew Decyzyjnych oraz klasyfikatora Naiwnego Bayesa.
* **Inżynieria cech:** Wybór najistotniejszych zmiennych (Feature Selection) i ich wpływ na dokładność modelu.
* **Ewaluacja:** Analiza porównawcza modeli na zbiorach treningowych i testowych pod kątem overfittingu.

## Główne wnioski:
* Najwyższą skuteczność (dokładność 0.9775) uzyskał model **Naiwnego Bayesa**.
* Wykazano odporność wybranych metod na "przekleństwo wymiarowości".

## Technologie:
* Język R
* Biblioteki do uczenia maszynowego i wizualizacji wyników.
