# Binary Tree Operations (C++)

Projekt demonstruje zaawansowane operacje na strukturach drzewiastych w języku C++. Zawiera autorską implementację dynamicznego drzewa binarnego wraz z algorytmami manipulacji całymi strukturami (dodawanie i odejmowanie drzew).

### Kluczowe Funkcjonalności:
* **Dynamiczne Zarządzanie Pamięcią:** Implementacja oparta na wskaźnikach z pełną kontrolą cyklu życia obiektów (usuwanie drzewa po zakończeniu operacji).
* **Aritmetyka Drzew:** Funkcje `dodajWezly` i `odejmijWezly`, które rekurencyjnie łączą dwie struktury w jedną nową na podstawie wartości węzłów.
* **Przeciążanie Operatorów:** Zaimplementowany `operator[]` pozwalający na dostęp do wartości węzłów w sposób zbliżony do tablic.
* **Wizualizacja Konsolowa:** Funkcja rysująca strukturę drzewa w terminalu z zachowaniem odpowiednich wcięć, co ułatwia debugowanie i analizę trajektorii wzrostu drzewa.
