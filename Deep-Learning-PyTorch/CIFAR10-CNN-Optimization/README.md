CIFAR-10 CNN Analysis

Projekt badający wpływ technik optymalizacji na skuteczność sieci konwolucyjnych (CNN), stworzony w ramach kursu Uczenie Maszynowe.

Kluczowe cechy:

-Modele: Porównanie architektury Baseline, Dropout, Batch Normalization oraz ich hybrydy.

-Środowisko: Wykorzystanie biblioteki PyTorch oraz akceleracji CUDA.

-Eksperyment: Analiza dokładności klasyfikacji obrazów na zbiorze CIFAR-10.

Projekt pokazuje praktyczne zastosowanie technik regularyzacji w głębokim uczeniu (Deep Learning) i ich bezpośredni wpływ na problem przeuczenia (overfitting).
