MNIST Neural Network from Scratch

Implementacja wielowarstwowej sieci neuronowej do rozpoznawania cyfr, stworzona w ramach kursu Uczenie Maszynowe.

Kluczowe cechy:

-Algorytm: Ręczna implementacja wstecznej propagacji błędu (Backpropagation).

-Struktura: Możliwość testowania różnych architektur (różna liczba neuronów).

-Od zera: Czysty Python i NumPy, samodzielne zarządzanie wagami i biasami.
