# Optymalizacja i Badania Operacyjne (Python)

Folder zawiera rozwiązania problemów optymalizacyjnych z wykorzystaniem programowania liniowego i biblioteki `PuLP`.

## Główne zagadnienia:
* **Problemy decyzyjne:** Maksymalizacja zysków przy ograniczonych zasobach (produkcja, czas pracy).
* **Minimalizacja kosztów:** Optymalny wybór dostawców oraz logistyka produktów.
* **Problemy klasyczne:** Implementacja problemu plecakowego (wybór produktów o największej wartości przy ograniczeniu wagowym).
* **Zarządzanie zapasami:** Modele minimalizujące straty surowców.

## Technologie:
* **Język:** Python
* **Biblioteki:** `PuLP` (Linear Programming), `pandas` (opcjonalnie do danych).

## Przykłady zastosowań:
1. **Optymalizacja produkcji:** Wyznaczanie idealnej ilości produktów (np. wypieków), aby zmaksymalizować marżę przy limitach mąki, masła i cukru.
2. **Problemy transportowe i finansowe:** Minimalizacja sumy przelewów między grupą osób lub kosztów transportu między punktami.
